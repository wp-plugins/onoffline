Notify your visitors when their internet connection goes down. Disable links so they don't end up with error pages.

== Description ==
Notify your visitors when their internet connection goes down. Disable links so they don't end up with error pages.

== Installation ==
Upload onOffline folder to your wp-content/plugins/ directory.
Activate the plugin through the 'Plugins' menu in WordPress.
Go to Settings > onOffline to set preferences.
Save, and it's done.
